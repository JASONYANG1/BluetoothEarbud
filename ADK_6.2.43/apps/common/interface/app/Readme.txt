common/interface/app

Header files in this directory ported from BlueCore as a part of an API
interface between firmware and VM Applications.

Currently only the USB audio class test device implementation in Hydra uses it
as it incorporates USB device class library from BlueCore VM Applications.

These files shall not be used in a common firmware code.
